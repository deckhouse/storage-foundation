## 002-fix-cve-golang-x.patch

Bumps `golang.org/x/net` -> `v0.55.0`, `golang.org/x/crypto` -> `v0.52.0`
and `golang.org/x/sys` -> `v0.45.0` in `go.mod` / `go.sum` to fix the
`golang.org/x/net/html`, `golang.org/x/crypto/ssh` and
`golang.org/x/sys/windows` CVEs reported by Trivy. Generated by cloning
the pinned upstream version, applying the existing patches, then
`go get`/`go mod tidy` and `git format-patch -1`; touches only
`go.mod` / `go.sum` (the build re-runs `go mod vendor`/`download`).

# Patches

Applied to `kubernetes-csi/external-provisioner` v5.3.0 (k8s 1.20
track) during build of the `csi-provisioner` binary.

## 001-fix-cve.patch

Bumps direct/indirect dependencies in `go.mod` / `go.sum` to versions
that fix CVEs reported by Trivy on the v5.3.0 binary:

- `google.golang.org/grpc` `v1.72.1 -> v1.80.0` (fixes CVE-2026-33186)
- `golang.org/x/crypto` `v0.38.0 -> v0.47.0`
- `google.golang.org/protobuf` `v1.36.6 -> v1.36.11`

OpenTelemetry is not flagged for v5.3.0 by Trivy, so it stays at
v1.39.0 after `go mod tidy -e`.

Generated by cloning v5.3.0, running `go mod edit -require=...` and
`go mod tidy -e`, then `git format-patch -1`. Only touches `go.mod` /
`go.sum`; the build pipeline does `rm -r vendor && go mod download`
before `make build`. Verified end-to-end inside `golang:1.25-alpine`
with `go build -mod=readonly`.

# Patches

Applied to `kubernetes-csi/external-attacher` v4.11.0 during build of
the `csi-attacher` binary.

## 001-fix-cve.patch

Bumps direct/indirect dependencies in `go.mod` / `go.sum` to versions
that fix CVEs reported by Trivy:

- `google.golang.org/grpc` `v1.72.2 -> v1.80.0` (fixes CVE-2026-33186)
- `go.opentelemetry.io/otel` `v1.38.0 -> v1.43.0` (fixes CVE-2026-29181)
- `go.opentelemetry.io/otel/sdk` `v1.36.0 -> v1.43.0` (fixes
  CVE-2026-24051, CVE-2026-39883)
- `golang.org/x/crypto` `v0.45.0 -> v0.47.0`
- `google.golang.org/protobuf` `v1.36.8 -> v1.36.11`

The patch was generated by cloning v4.11.0, running
`go mod edit -require=...` for the three core modules above followed by
`go mod tidy -e` against `GOPROXY=proxy.golang.org`, then
`git format-patch -1`. It only modifies `go.mod` / `go.sum`; the build
pipeline does `rm -r vendor && go mod download` before `make build`,
so the new versions are pulled fresh from the module proxy. Verified
end-to-end inside `golang:1.25-alpine` with `go build -mod=readonly`.
